# NFT Airdrop Input File

## Overview

This guide will help you format an Input file for an [NFT airdrop](../airdrops/nfts-to-users). This works best for airdrops of the same NFT to many users.

<Callout type="info" emoji="ðŸ“˜">
  Feel free to edit the file after creation or combine multiple files together.
</Callout>

## Requirements ðŸ“‹

- Maize Software: Make sure it's [downloaded](../download-maize) and [setup](../setup-maize) correctly.
- Recipient Details: This includes both wallet addresses and/or ENS addresses.
- NFT Information: Data related to the NFT you plan to distribute.
- Transfer Details: Specify the transfer amount and any accompanying memo or note.

## Steps

<Steps>
  ### Initiate `NFT Airdrop Input File`
  Navigate to the Scripting Section within the Maize Menu. Here, you'll find the NFT Airdrop Input File option. Click on it to start.

  ### Enter Recipient Details
  Inside the large textbox, input both wallet and ENS addresses. Ensure each address occupies its own line.

![Input addresses](/images/tutorials/nftAirdropInputFile/Addresses.jpg)

  ### Enter NFT Information
  Insert the data associated with your NFT into the second textbox. If you're uncertain, this data can be retrieved from any of the [lookups](../lookups).
    - Example: 0x11fe19d73d3ce532b4398b85f7f72fb309720e853fce8f5f374ea707d46448ae

  ### Enter Transfer Details
- In the third textbox, specify the amount you intend to transfer to each wallet.
- In the final textbox, input the memo or note you wish to accompany each transfer.

![Input Transfer Information](/images/tutorials/nftAirdropInputFile/Information.jpg)

  ### Initiate the Creation
  Once you're confident with the entered data, click on 'Create'. Allow the application some time to process and complete its task.

  ### Review the Result
  Upon completion, the application will automatically open the generated file. Review its content to ensure everything is as intended. You can now use this file for your [NFT airdrop](../airdrops/nfts-to-users).

![Input Transfer Information](/images/tutorials/nftAirdropInputFile/Final.jpg)

  ### Congratulations! ðŸŽ‰
You have successfully crafted an Input file for your crypto airdrop. Well done!
  </Steps>
